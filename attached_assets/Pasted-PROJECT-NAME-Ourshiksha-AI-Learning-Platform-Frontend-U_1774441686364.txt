PROJECT NAME: Ourshiksha AI Learning Platform (Frontend UI)

GOAL:
Build a modern, premium AI-powered e-learning platform UI using React (Vite) + Tailwind CSS. The design must follow a soft pastel blue-purple theme and feel like a mix of Coursera + Notion + modern SaaS dashboards.

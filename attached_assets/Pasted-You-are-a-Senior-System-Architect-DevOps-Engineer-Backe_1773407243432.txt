# Required Features

1. Video Upload API

Teachers should be able to upload MP4 videos.

Endpoint

POST /upload-video

Functionality

• Accept MP4 upload
• Store temporarily
• Send file to video processor

---

2. Video Processing

Use FFmpeg to convert MP4 into HLS format.

Generate

.m3u8 playlist
.ts video segments

Example command

ffmpeg -i input.mp4 
-c:v libx264 
-c:a aac 
-hls_time 10 
-hls_list_size 0 
-f hls output.m3u8

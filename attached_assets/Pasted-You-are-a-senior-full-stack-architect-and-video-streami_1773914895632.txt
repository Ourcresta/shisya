Your task is to design and implement a complete working system with clean, production-ready code.

---

## 🎯 SYSTEM GOAL

* Store videos cheaply
* Stream videos globally with low buffering
* Minimize bandwidth cost
* Support scaling from 100 → 10,000 users

---

## 🧠 ARCHITECTURE (STRICT)

* Storage: Backblaze B2 (store HLS files)
* CDN: Bunny CDN (pull zone connected to B2)
* Streaming format: HLS (.m3u8 + .ts segments)
* Frontend: React (or simple HTML if needed)
* Backend: Node.js (Express)

IMPORTANT:

* Do NOT use Cloudflare for video delivery
* Cloudflare can be used only for website security (optional)

---

Generate FFmpeg command for:

* 720p optimized compression
* HLS playlist (.m3u8)

* reduced size (~300MB per video)

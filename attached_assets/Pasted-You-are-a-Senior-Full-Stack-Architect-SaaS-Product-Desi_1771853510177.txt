PART 1: UDYOG INTERNSHIP SYSTEM
---------------------------------------------------

CORE CONCEPT:
Internships are batch-based industry simulations.

Each Internship Project runs in batches of 5 students.

---------------------------------------------------
INTERNSHIP FLOW
---------------------------------------------------

STEP 1 – Eligibility Check
- Student must complete required course OR pass skill assessment.
- Skill score stored in database.

STEP 2 – Batch Formation
- Once 5 eligible students enroll → create new batch.
- Automatically assign roles based on skill score:

Highest skill → Team Lead
Next 3 → Developers
Lowest → QA/Tester

Batch status = "Active"

STEP 3 – Project Allocation
Admin creates Internship Template with:
- Title
- Description
- Required Skills
- Duration (2–4 weeks)
- Milestones
- Task Templates

Each batch gets its own instance of tasks.

STEP 4 – Internship Workspace (Mini Company Simulation)

Each Batch Dashboard must include:

- Project Overview
- Team Members & Roles
- Kanban Task Board (Todo / In Progress / Review / Completed)
- Individual Assigned Tasks
- Submission Upload
- AI Mentor Placeholder
- Deadline Countdown
- Progress Percentage
- Performance Score

---------------------------------------------------
TASK SYSTEM
---------------------------------------------------

Tasks must support:

- CRUD
- Assigned To
- Status
- Due Date
- File Submission
- Admin Feedback
- Score per task

---------------------------------------------------
EVALUATION SYSTEM
---------------------------------------------------

Final Performance Score calculated based on:

- Task Completion Rate
- Deadline Compliance
- Task Quality Score
- Team Collaboration Score (admin input)

Store performance_score in BatchMembers table.

---------------------------------------------------
CERTIFICATION SYSTEM
---------------------------------------------------

If:
- All tasks completed
- Admin approves final submission

Then:
- Generate Internship Completion Certificate record
- Include:
   - Student Name
   - Internship Title
   - Role
   - Batch ID
   - Performance Score
   - Duration
- Enable PDF download (placeholder allowed)

PART 2: HR HIRING SYSTEM
---------------------------------------------------

HR users represent verified companies.

HR must be approved by Admin before accessing dashboard.

---------------------------------------------------
HR CAPABILITIES
---------------------------------------------------

1. Post Job
Fields:
- Job Title
- Description
- Required Skills
- Internship Required? (Yes/No)
- Minimum Skill Score
- Location
- Salary Range
- Job Type
- Deadline

2. View Candidates

HR can filter students by:
- Skill Score
- Internship Completed
- Performance Score
- Specific Skills
- Role (Team Lead / Developer)

3. AI Matching Logic

When job is posted:
Calculate matching_score:

matching_score =
( skill_match_weight +
 internship_performance_weight +
 certification_weight ) / 3

Show HR:
Top 10 Recommended Candidates automatically.

4. Manage Applications
- Applied
- Shortlisted
- Rejected
- Hired

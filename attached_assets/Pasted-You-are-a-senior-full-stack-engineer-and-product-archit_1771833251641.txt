1️⃣ Udyog Main Page

   - Industry Certification

2️⃣ Virtual Internship System

Internship must work like this:

STEP 1: Skill Evaluation
- Student takes AI skill assessment
- Store score in database

STEP 2: AI Internship Assignment
- Based on skill level:
    Beginner → Junior Intern Project
    Intermediate → Project Associate
    Advanced → Lead Developer Simulation
- Assign project automatically

STEP 3: Internship Dashboard

Intern Dashboard Includes:
- Assigned Project
- Project Description
- Deliverables Checklist
- Submission Section
- AI Mentor Chat (placeholder UI)
- Progress Bar
- Team View (simulated teammates)
- Deadline Countdown

3️⃣ Project Management Simulation

Each internship must have:
- Tasks (CRUD functionality)
- Status: Todo | In Progress | Review | Completed
- Submission upload
- AI feedback placeholder

4️⃣ Certification Logic

If:
- All tasks completed
- Submission approved

Then:
- Generate Internship Completion Certificate
- Store certification in database
- Show downloadable PDF button (placeholder logic allowed)

5️⃣ Admin Panel (Basic)

Admin must be able to:
- Create Internship Projects
- Define skill level required
- View student submissions
- Approve / Reject

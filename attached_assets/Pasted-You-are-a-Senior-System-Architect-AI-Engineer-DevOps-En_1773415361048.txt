# Step 1 — Generate Adaptive HLS Streaming

Use FFmpeg to generate multiple video qualities.

Qualities

```
1080p
720p
480p
360p
240p
```

Example FFmpeg command

```
ffmpeg -i lesson.mp4 \
-map 0:v -map 0:a \
-s:v:0 1920x1080 -b:v:0 5000k \
-s:v:1 1280x720 -b:v:1 2800k \
-s:v:2 854x480 -b:v:2 1400k \
-s:v:3 640x360 -b:v:3 800k \
-s:v:4 426x240 -b:v:4 400k \
-c:v libx264 \
-c:a aac \
-f hls master.m3u8
```

This generates

```
master.m3u8
1080p.m3u8
720p.m3u8
480p.m3u8
360p.m3u8
240p.m3u8

segment0.ts
segment1.ts
segment2.ts
```

# Step 7 — Multi Language Audio in HLS

Create separate audio tracks

```
audio_en.m3u8
audio_hi.m3u8
audio_ta.m3u8
audio_te.m3u8
```

Attach to master playlist.

Example master.m3u8

```
#EXTM3U

#EXT-X-MEDIA:TYPE=AUDIO,GROUP-ID="audio",NAME="English",LANGUAGE="en",URI="audio_en.m3u8"
#EXT-X-MEDIA:TYPE=AUDIO,GROUP-ID="audio",NAME="Hindi",LANGUAGE="hi",URI="audio_hi.m3u8"
#EXT-X-MEDIA:TYPE=AUDIO,GROUP-ID="audio",NAME="Tamil",LANGUAGE="ta",URI="audio_ta.m3u8"

#EXT-X-STREAM-INF:BANDWIDTH=800000,AUDIO="audio"
video.m3u8
```

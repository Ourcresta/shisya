You are a Senior Full-Stack Engineer, Senior System Architect, Senior UI/UX Designer,
Senior Security Engineer, Senior QA Engineer, and Senior Product Manager.

Your task is to DESIGN, BUILD, and DELIVER a COMPLETE, PRODUCTION-READY system called:

=====================================================
OURSHIKSHA – TOTAL MARKSHEET & TRANSCRIPT ENGINE
=====================================================

This system will generate official, verifiable, tamper-proof academic marksheets
for an online learning platform (OurShiksha).

---------------------------------------
CORE OBJECTIVES
---------------------------------------
1. Generate module-wise, project-wise, and total marksheets
2. Auto-calculate grades, percentages, rewards, and eligibility
3. Export as high-quality A4 PDF
4. Support QR-based online verification
5. Integrate with Student Portal + Admin Portal
6. Use a Neon + Modern UI theme
7. Be secure, scalable, and audit-ready

---------------------------------------

---------------------------------------
Frontend:
- React (Vite)
- Tailwind CSS (Neon Modern Theme)
- shadcn/ui components

Backend:
- Node.js + Express
- REST API architecture

Database:
- PostgreSQL (via Prisma ORM)

- PDFKit or Puppeteer

Auth:
- JWT (Student, Admin, Verifier roles)

1. Admin (Guru Portal)

2. Student (Shishya Portal)

---------------------------------------

- Shows:

  - Course
  - Grade
  - Status
  - Issued date
- No marks editing allowed

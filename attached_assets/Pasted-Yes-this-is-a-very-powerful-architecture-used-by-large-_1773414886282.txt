# 3️⃣ Step 2 — Adaptive Streaming (Multi Quality)

Generate multiple resolutions so slow internet users can watch.

Qualities:

```
1080p
720p
480p
360p
240p
```

FFmpeg example:

```
ffmpeg -i lesson.mp4 \
-map 0:v -map 0:a \
-s:v:0 1920x1080 -b:v:0 5000k \
-s:v:1 1280x720  -b:v:1 2800k \
-s:v:2 854x480   -b:v:2 1400k \
-s:v:3 640x360   -b:v:3 800k \
-s:v:4 426x240   -b:v:4 400k \
-f hls master.m3u8
```

This creates:

```
master.m3u8
1080p.m3u8
720p.m3u8
480p.m3u8
360p.m3u8
240p.m3u8
```

Player automatically switches quality.
